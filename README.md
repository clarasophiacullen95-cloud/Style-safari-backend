## Project Overview

This backend:

1. Fetches products from **Base44 ProductFeed**.  
2. Stores products in **MongoDB Atlas**.  
3. Provides endpoints to **sync, search, and fetch products**.  
4. Runs on **Vercel serverless functions**.
