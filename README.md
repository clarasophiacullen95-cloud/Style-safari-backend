# Affiliate Product Search App

## Description
This app allows users to search for products across multiple affiliate websites and automatically updates its product database via affiliate APIs. It supports filtering by category, price, and rating, providing a fast, centralized product search experience.

---

## Features
- Automatic product uploads from affiliate APIs
- Multi-website product search
- Filter products by:
  - Category
  - Price range
  - Rating
- Backend built with serverless functions (Vercel compatible)
- Easy deployment with Git + Vercel
